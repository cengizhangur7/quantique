# Quantique

**Yapay Zeka Destekli Film ve Dizi Öneri Platformu**

##  Proje Açıklaması

Quantique, kullanıcıların film ve dizi tercihlerini yapay zeka destekli sohbet arayüzü üzerinden öğrenen ve kişiselleştirilmiş öneriler sunan modern bir platform projesidir. Kullanıcılar, doğal dil işleme teknolojileri sayesinde sohbet ederek tercihlerini belirleyebilir ve kendilerine özel film/dizi önerileri alabilirler.

##  Temel Özellikler

- **Kullanıcı Kaydı ve Profil Yönetimi**
- **Yapay Zeka Destekli Sohbet Tabanlı Tercih Belirleme**
- **Detaylı Filtreleme ve Keşif Özellikleri**
- **Akıllı Öneri Motoru**
- **Kullanıcı Paneli** (İzleme listesi, geçmiş, puanlamalar)
- **Sosyal Özellikler** (gelecek güncellemeler)
- **Oyunlaştırma Sistemi** (rozetler ve hedefler)

##  Teknoloji Yığını

### Frontend
- **Next.js** (React Framework)
- **Vercel** (Hosting)

### Backend
- **FastAPI** (Python)
- **NLTK/spaCy** (Doğal Dil İşleme)
- **Render/Railway** (Hosting)

### Veritabanı & Kimlik Doğrulama
- **Supabase** (PostgreSQL + Auth)

### Veri Kaynağı
- **TMDB API** (The Movie Database)

##  Kurulum

```bash
# Projeyi klonlayın
git clone https://github.com/cengizhangur7/quantique.git
cd quantique

# Gerekli bağımlılıkları yükleyin
# Frontend bağımlılıkları
npm install

# Backend bağımlılıkları
pip install -r requirements.txt
```
